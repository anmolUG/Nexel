# Nexel

Nexel is an instant edge-deployment platform architected seamlessly for full-stack React and Next.js applications. Connect your GitHub repository to securely spin up an automated Docker pipeline that orchestrates, compiles, and deploys your code directly to the edge network in seconds. 

## Features
- **One-Click Deploy**: Push your application to a connected branch to watch it automatically build and deploy within seconds.
- **Isolated Build Servers**: Highly reliable Dockerized build pipelines securely compile environments on AWS ECS.
- **Real-time Kafka Streaming**: Automated websocket channels securely pipe build logs directly to the integrated developer dashboard.
- **S3 Reverse Proxy Edge Network**: Ultra-fast caching and content delivery dynamically served across cities globally.

## Tech Stack
- Next.js (App Router)
- React Three Fiber
- Framer Motion
- Node.js & Docker
- Apache Kafka & Redis
- AWS (ECS, S3, EC2)
